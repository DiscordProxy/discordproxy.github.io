Use

Send an HTTP/Post request to https://discordproxy.github.io/payload.php

Params

webhook_id your discord webhook id (should be everything after discordapp.com/webhooks/) e.g. 782193193801289321/HUAJDHI9910Ujia9-hiha819298diaujiluj_hau28o1gh_ua2hda9-182uan3

payload the data for the webhook, as a json string
